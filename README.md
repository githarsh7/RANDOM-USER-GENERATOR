# Random User Generator

A responsive web application that fetches and displays random user profiles using the Random User API. Built with HTML, CSS, JavaScript, and jQuery, featuring animated backgrounds, glassmorphism UI, and smooth interactive effects.

---

## Features

- Generate random user profiles instantly
- Display user image, name, email, and location
- Animated gradient background
- Modern glassmorphism card design
- Smooth animations and hover effects
- Responsive layout for mobile devices
- Integration with the Random User API

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- Random User API
